2. Declaration Block:A declaration block contains one or more declarations separated by semicolons (;).

Each declaration includes a CSS property name followed by a colon (:) and a value.

Declaration blocks are enclosed in curly braces ({}).


3. Properties and Values: CSS properties define what aspect of the selected elements you want to style, and values specify how you want to style them.



Note!!! The link element with the rel attribute is important in HTML because it defines the relationship between the current document and an external resource. It is commonly used to link stylesheets, icons, and other resources to an HTML document. The rel attribute specifies the type of relationship between the current document and the linked resource, and it helps browsers and other user agents understand how to handle the linked resource.

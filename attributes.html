HTML ATTRIBUTES

- TML attributes provide additional information about HTML elements.

- All HTML elements can have attributes

- Attributes provide additional information about elements

- Attributes are always specified in the start tag

- Attributes usually come in name/value pairs like: name="value"

1. src (Source):

- Specifies the source URL of an external resource, such as an image or script.

Example:

<img src="image.jpg" alt="Description">


2. href (Hypertext Reference):

- Specifies the URL of a linked resource, such as a webpage or a file.

Example:

<a href="https://www.example.com">Visit Example.com</a>


Visit Example.com

3. alt (Alternative Text):

- Provides alternative text for images, which is displayed if the image cannot be loaded.

Example:

<img src="profile.jpg" alt="Profile Picture">


4. width and height

- Specifies the width and height of an element, such as an image or table cell.

Example:

<img src="picture.jpg" alt="Picture" width="200px" height="200px">


5. class:

- Assigns a class to an HTML element, allowing the element to be styled with CSS or selected with JavaScript.

Example:

<p class="highlight">This paragraph has a special style.</p>


This paragraph has a special style.

6. id:

- Assigns a unique identifier to an HTML element, which can be used for styling or scripting purposes.

Example:

<div id="header">This is the header.</div>


This is the header.

7. style:

- Defines inline styles for an element using CSS properties.

Example:

<p style="color: blue; font-size: 16px;">Styled Text</p>


Styled Text

8. placeholder:

- Provides a short hint or example text in an input field before the user enters a value.

Example:

<input type="text" placeholder="Enter your name">


9. disabled:

- Disables an input, button, or other interactive element, preventing user interaction.

Example:

<button disabled>Disabled Button</button>


10. checked:

- Specifies that a checkbox or radio button should be pre-selected (checked) when the page loads.

Example:

<input type="checkbox" checked> Remember me


Remember me
